# Video Search

### git 으로 협업하기

1. git clone https://github.com/hyunkyung12/video-searching.git
2. git checkout -b (branch 이름)
3. git add (수정내용)
4. git commit -m (메시지)
5. git push origin (branch 이름)
6. 다른 사람의 내용을 적용하려면 git pull origin (branch 이름)
